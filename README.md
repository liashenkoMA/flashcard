# Flashcards JP

Веб-приложение для изучения японского языка с помощью флеш-карточек.

Проект создан как полноценная full-stack платформа с клиентской и серверной частью:
пользовательская система, личный кабинет, управление учебными материалами и механизм повторения.

## О проекте

Flashcards JP помогает изучать:

- японскую азбуку (Hiragana / Katakana)
- кандзи
- японские слова

Основная идея проекта — использовать формат карточек для постепенного закрепления знаний через повторение.

## Возможности

### Пользовательская часть

- регистрация и авторизация
- личный кабинет
- управление профилем

### Обучение

- интерактивные флеш-карточки
- система повторения карточек
- отслеживание веса изучаемых элементов
- отдельные модули для разных разделов языка

### Контент

- добавление и управление кандзи
- добавление слов
- работа с японской письменностью

## Архитектура проекта

Проект разделен на две основные части:

```
flashcards-jp
│

│

```

## Статус проекта

Проект находится в активной разработке.

Текущая версия представляет собой открытую базовую платформу, включающую основную архитектуру приложения и учебный функционал.

Дальнейшее развитие продукта включает дополнительные возможности и интеграции, которые будут развиваться отдельно от публичной версии.

## Цель проекта

Проект разработан как практическая full-stack разработка с использованием современных инструментов экосистемы JavaScript/TypeScript.

Основные задачи:

- построение масштабируемой архитектуры
- работа с frontend и backend одновременно
- разработка API
- проектирование базы данных
- создание пользовательского продукта от идеи до deployment
